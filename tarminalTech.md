# ターミナルのまとめ

## 小技
```
 $ ctr + r    # 過去に入力したコマンドを検索する
 $ ctr + l    # ターミナルを画面分移動（clear）
 $ ctl + d    # ターミナルを閉じる（exit）
```

## カーソル移動
```
 $ ctl + a    # カーソルを先頭に移動
 $ ctl + e    # カーソルを行末に移動
 $ ctl + f    # カーソルを1つ進める
 $ ctl + b    # カーソルを1つ戻す
```

## Macターミナルの機能
```
 $ cmd + f    # 表示されているターミナルテキストの中から検索
```
